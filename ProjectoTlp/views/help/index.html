Ajuda - Marcação de Consulta

Como marcar uma consulta?

Para marcar uma consulta, siga os seguintes passos:

  1. Acesse a página de marcação de consulta
  2. Selecione a consulta desejada
  3. Selecione a data e horário disponíveis
  4. Preencha seus dados pessoais
  5. Clique em "Agendar consulta"

Posso cancelar uma consulta?

Sim, você pode cancelar uma consulta a qualquer momento. Para cancelar, acesse a página de consulta e clique em "Cancelar consulta".

Preciso chegar com antecedência para minha consulta?

Recomendamos que você chegue com pelo menos 15 minutos de antecedência para sua consulta.
